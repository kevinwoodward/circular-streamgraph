Documentation:


Features:

-Select a preformatted dataset to display, or a random generation of a monthly dataset (uploads don't work due to issues with cross origin file errors)
-Use inner radius and outer radius sliders to adjust the size of the streamgraph
-Use the SVG dimensions slider to resize the group of SVG layers for better visibility
-Like a streamgraph, the data of the same time value stacks along the axes
-Color layers represent a cycle of data
-If the data is in weeks or months (e.g. the data points per key is 7 or 12), the axes will reflect that. Otherwise a number will simply be displayed.

Notes:


-Tested in Chrome only, theoretically should work in most browsers.
-Using custom data sets. Loading files, referencing files, or manipulating files all throw Cross Origin Reference security errors, which is the reasoning behind the predetermined datasets.
-Screen flashing on redraw. This is due to the fact that the entire script reruns and removes all the SVG elements for a redraw
